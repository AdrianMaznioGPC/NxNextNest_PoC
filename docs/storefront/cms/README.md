# Storefront CMS Domain

## Purpose
Renders CMS-style blocks on the storefront home and content pages.

## Key Files
- `apps/storefront/components/cms/block-registry.tsx`
- `apps/storefront/components/cms/block-renderer.tsx`
- `apps/storefront/components/cms/blocks/*`

## Inputs And Outputs
- Inputs: CMS block arrays from page data
- Outputs: rendered block trees
